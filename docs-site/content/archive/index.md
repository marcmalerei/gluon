# Release archive

Documentation is retained by supported release line. The version selector and
this archive resolve to immutable version paths rather than a floating page.

## Supported

- [Gluon 1.0.7 documentation](/gluon/1.0.7/) — current supported release line.

## Archived

- Gluon 1.0.6 — first supported release, retained in repository history.

The unpublished `0.0.0` prototype was never a supported release line; its
documentation remains available in repository history. When a supported line
leaves support under ADR 0002, its content remains addressable from this archive
unless a legal or security requirement mandates removal.
