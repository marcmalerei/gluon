## Component libraries

Gluon 1.2 adds a public Quarks manifest and loader contract for separately
packaged component libraries. Consumers request declared entries explicitly,
retain observable loading and cache state, and own Custom Element registration,
constructable stylesheets, SSR style snapshots, and disposal. The maintained
package consumer and Storybook catalog exercise those same public boundaries.

[Build component boundaries →](/gluon/1.4.0/guides/components/)
