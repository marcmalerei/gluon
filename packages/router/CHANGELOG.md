# Changelog

All notable changes to `@gluonjs/router` are recorded here.

## [Unreleased]

### Added

- Public API purpose, parameter, failure, ownership, and curated usage
  documentation for `createMemoryHistory()`.
- Browser, hash, and DOM-free memory histories with state positions and saved
  scroll coordinates.
- Static, dynamic, custom-pattern, optional, repeatable, nested, named,
  redirected, and aliased route matching.
- Typed named locations, deterministic query serialization, navigation guards,
  explicit failures, lazy route components, and server snapshots.
- `RouterLink`, `RouterView`, application plugin, and injection helpers.
